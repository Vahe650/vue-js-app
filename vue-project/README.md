# vue-project

## Project Setup

```sh
npm install
```

## Server Setup
```sh
npm install -g json-server
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Start local server

```sh
npm run server
```
local server in port http://localhost:3000

### For login  

```sh
for User -> login: user, password: user
```
